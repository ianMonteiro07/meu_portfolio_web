# 👨‍💻 Meu Portfólio Web

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido como atividade prática para a disciplina de **Desenvolvimento Web** do curso de **Ciência da Computação (UNIMA)**.

O objetivo do projeto é aplicar os conceitos de estruturação web, design responsivo e controle de versão, criando uma página profissional para exibir minhas habilidades e projetos.

## 🎨 Funcionalidades e Design

O projeto foi construído com um tema **"Dark Tech"** moderno e inclui:
- **HTML Semântico:** Estruturação correta utilizando tags como `<header>`, `<section>`, `<footer>`, etc.
- **CSS Avançado:** Uso de variáveis (`:root`), flexbox e pseudo-elementos (`::after`, `::selection`).
- **Responsividade Total:** Interface adaptável para dispositivos móveis e desktops utilizando *Media Queries* e unidades de medida relativas (`rem`, `%`).
- **Micro-interações:** Efeitos de *hover*, transições suaves e sombras dinâmicas (sem uso de JavaScript).

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído com as seguintes tecnologias:

* **HTML5**
* **CSS3**
* **Git e GitHub** (Versionamento e Deploy)
